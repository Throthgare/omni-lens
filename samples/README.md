# OmniLens Sample Reports

This folder contains sample output files generated by OmniLens to demonstrate its capabilities.

## Files

- `sample_report.html` - Interactive HTML report showing analysis of a test repository
- `sample_report.json` - JSON output with structured data
- `sample_report.md` - Markdown summary report

## How to Generate Your Own Reports

```bash
# Analyze current directory
omnilens

# Generate HTML report
omnilens --format html --output report.html

# Generate JSON data
omnilens --format json --output report.json

# Analyze with tech debt metrics
omnilens --tech-debt --complexity --format html
```

## Features Demonstrated

- **Git Analysis**: Commit history, author statistics, category breakdown
- **Code Metrics**: File counts, lines of code, language detection
- **Interactive Charts**: Visual representation of data using Chart.js
- **Responsive Design**: Works on desktop and mobile devices
- **Professional Styling**: Clean, modern interface with Tailwind CSS

Open `sample_report.html` in your browser to see the full interactive report!
